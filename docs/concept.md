# Communication Compiler Concept

The product is a Communication Compiler, not a generic slide generator.

It turns rough user intent into a structured business communication artifact:

```text
Free Talk
  -> Claim Candidates
  -> 4-Line Message Kernel
  -> Diagnosis
  -> Evidence / Gaps / Objections
  -> Message-First Deck and Brief
```

The five-page deck is the final presentation UI. The core artifact is the 4-Line Message Kernel:

1. Audience Premise
2. Complication / Gap
3. Core Claim
4. Requested Decision / Action

The Agent should not immediately create slides. It first identifies what the user may actually want to say, sharpens the message, and shows where the message is weak or unsupported.

## Product Narrative

AI can already generate slides. In real business communication, the hard part is not drawing slides. The hard part is deciding what should be said, why it matters now, what evidence supports it, and what decision the audience should make.

The Communication Compiler makes this invisible thinking process visible.
